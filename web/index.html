Nuvem de palavras dos planos de governo* das eleições de 2022

*somente Governo/Presidencia fazem planos de governo

Perguntas frequentes

1. Qual é a fonte dos dados?

Planos de governo disponibilizados pelo site do TSE.

2. Como são criadas as nuvems de palavras?

Para cada plano de governo, extraio os textos do arquivo .pdf e depois gero a nuvem utilizando com base nas palavras mais frequentes de cada programa. Esse processo é feito com a linguagem de programação Python e as seguintes bibliotecas foram utilizadas(PyPDF2 e wordcloud)

3. Alguma palavra foi excluída da análise?

Sim, todas essas palavras foram removidas dos programas de governo antes da geração da nuvem.

4. Por que algumas nuvems estão com palavras incompletas?

Por dois principais motivos, o arquivo PDF foi gerado de uma forma que os textos não são facilmente identificados pelo biblioteca que utilizei ou o arquivo PDF foi gerado como imagem e não como texto. Não fiz nenhum tipo de validação pra garantir que todas palavras do PDF estão sendo lidas corretamente, portanto não tirem muitas conclusões desses nuvems, pois é um projeto nada acadêmico, somente um hobbie. 🤓

5. Onde encontro mais detalhes do projeto?

No repositório do github.com/wxwr/wordcloud_govplans_elections2022

6. Por que a nuvem de palavras não foi gerada para alguns candidatos?

Diversos motivos, por exemplo, o arquivo do programa de governo é a carteira de motorista do candidato🤣, o PDF não permite fazer cópia do texto, o arquivo PDF está em má qualidade ou é uma imagem, etc. Segue a lista desses candidatos/arquivos:
